# Portfólio Pessoal

Portfólio moderno e responsivo desenvolvido com React, Tailwind CSS e Vite.

## Tecnologias

- React 18
- React Router DOM
- Tailwind CSS
- Lucide React (ícones)
- Framer Motion (animações)
- Vite

## Instalação

```bash
# Instalar dependências
npm install

# Executar em modo desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build
npm run preview
```
